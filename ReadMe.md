GYM Geek is an Exercise tracker app built using the MERN stack.

## Technologies Used
* Node.js
* MongoDB
* Mongoose
* Express
* React.js
* JavaScript CSS, HTML5, 
* JWT
* Bcrypt
* Git + Github
* Bootstrap CSS Framework
